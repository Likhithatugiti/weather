Weather Details

About

This application provides current weather information for various cities.

Key Features:

City Search Feature: Users can enter a city name to get its weather information.

Current Weather Display: The app shows crucial weather details like temperature, humidity, wind speed, and a brief description of the weather.

Temperature Unit Switch: Users can toggle between Celsius and Fahrenheit for temperature readings.

Responsive Layout: The application is optimized for multiple devices, providing a consistent user experience on desktops, tablets, and smartphones.
